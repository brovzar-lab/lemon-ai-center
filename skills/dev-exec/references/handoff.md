# Pre-Production Handoff

## Readiness Criteria

Confirm ALL before handoff:
- Green Light Score 55+/75
- All genre conventions and obligatory scenes present
- Protagonist active with clear, castable arc
- Project Identity Statement fully realized on the page
- Budget tier confirmed, commercial math works
- No structural problems remain (character/dialogue polish may continue)
- A director or star can read this and say yes

Do not hand off a script that needs another development pass.

## Handoff Deliverables

### 1. Final Screenplay
Mark: draft number, date, "Development Draft — Approved for Pre-Production."

### 2. Project Identity Statement (Updated)
```
Title / Genre / Tone / Audience / Emotional Promise / Comparable Films
[One paragraph: what this movie IS at its best.]
```

### 3. Development History Brief
```
PROJECT / WRITER / DEVELOPMENT PERIOD / PASSES COMPLETED

INTAKE: [1-2 sentences on original submission]

JOURNEY:
Pass 1: [Focus — what changed and why]
Pass 2: [Focus — what changed and why]
Pass 3: [Focus — what changed and why]

KEY DECISIONS: [Decision — why made]
REJECTED APPROACHES: [Approach — why it didn't work]
NOTES FOR DIRECTOR: [What to protect, what has room for interpretation]
```

### 4. Character Profiles (Per Major Role)
```
Character / Role / Enneagram Type (if typed)
Arc Summary: [Lie, Want, Need, transformation]
Defining Trait: [What makes them castable]
Showcase Scene: [Page #]
ACTOR Test: [A/C/T/O/R scores]
Casting Notes: [Age range, type, specific actors discussed]
```

### 5. Budget Top Sheet
```
Title / Budget Tier / Estimated Total (MXN)
Key Budget Drivers: [What makes this expensive/cheap]
Cost Reduction Options: [Where it can give without hurting story]
Shooting Days Estimate: [Range]
```

### 6. Commercial Positioning Memo
```
Title / Genre / Target Market / Target Audience
LOGLINE: [1-2 sentences]
COMPS: Tone [film] / Structure [film] / Market [film]
SCORECARD: [/50]
MARKETING: Primary hook / Secondary / Social
POSTER CONCEPT: [One-sentence key image]
TAGLINE OPTIONS: [2 options]
FESTIVAL STRATEGY: [If applicable]
DISTRIBUTION: [Theatrical / Streaming / Hybrid]
INTERNATIONAL: [How it travels]
```

## Handoff Meeting Agenda

1. Project Identity Statement walkthrough — align everyone
2. Development history — what was tried and rejected (prevent reinventing rejected wheels)
3. Strengths to protect through production
4. Known soft spots with room for improvement
5. Character and casting discussion
6. Budget reality check
7. Commercial positioning and marketing angles
8. Timeline and production milestones
