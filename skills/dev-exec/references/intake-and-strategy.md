# Intake Memo & Development Strategy Templates

## Intake Memo Template

```
PROJECT:          [Title]
SOURCE:           [Spec Script / Pitch / Treatment / Book / Writer Idea]
WRITER:           [Name]
DATE:             [Date]

CONCEPT (One Sentence):
[If this can't be stated in one sentence, that's a red flag.]

THE 5-MINUTE READ:
  Genre:            [External] / [Internal]
  Genre Promise:    [Delivers? Y/N + why]
  Audience:         [Who specifically?]
  Would They Pay:   [Y/N + why]
  Star Role:        [Y/N + describe]
  ACTOR Test:       Agency [1-5] | Complexity [1-5] | Transformation [1-5] | Opportunities [1-5] | Range [1-5]
  Budget Tier:      [Bajo <6M MXN / Medio ~40M MXN / Alto >80M MXN]
  Commercial Ceiling: [Realistic estimate]
  Budget/Ceiling Match: [Y/N]

COMPARABLE FILMS:
  1. Tone:      [Film] — [Performance]
  2. Structure: [Film] — [Performance]
  3. Market:    [Film] — [Performance]

STRENGTHS:
  • [Specific — what works and why]
  • [Specific]

CONCERNS:
  • [Specific diagnosis — what's broken and why]
  • [Specific]

CLASSIFICATION: [ ] Green Light Track  [ ] Development Bet  [ ] Incubation  [ ] Pass

REASONING: [2-3 sentences]

RECOMMENDED NEXT STEP: [One specific action]
```

## Intake Decision Tree

```
NEW PROJECT → Can you state concept in one sentence?
  NO → Ask writer to clarify
  YES → Does it deliver on genre promise?
    NO → Is concept fixable? → NO: PASS / YES: DEVELOPMENT BET
    YES → Clear audience who would pay?
      NO → PASS or INCUBATE
      YES → Budget math works?
        NO → Adjustable? → NO: PASS / YES: DEV BET
        YES → GREEN LIGHT TRACK or DEV BET (based on script quality)
```

## Classification Guidelines

**GREEN LIGHT TRACK requires ALL of:** concept works in one sentence, genre promise clear, audience identifiable, star role exists, budget/commercial math works, script functional (fixable in 1-2 passes).

**DEVELOPMENT BET requires:** strong concept OR strong execution (at least one), identifiable solvable problems, commercial potential justifies investment.

**INCUBATION:** great concept but wrong timing, interesting writer with undeveloped idea, IP needing right angle.

**PASS:** concept doesn't survive scrutiny, no audience, budget mismatch with no fix, execution gap too large.

## Common Priority Sequences

**Broken Engine:** Concept/engine → Protagonist Want/Need → Structure → Character depth → Polish

**Passive Protagonist:** Moral blind spot + immoral effect → Opposition → Offer-refusal loops in Act 2 → Supporting cast → Genre obligations

**Genre Misfire:** Genre audit (missing conventions) → Core event = climax → Value progression → Tone alignment → Commercial sharpening

**Good Script Not a Movie:** Visual promise → Stakes/urgency → Set pieces → Star role → Marketability

**Development Bet Special:** Identify exceptional element → Rebuild weakest around it → Pressure-test survival → Commercial check
