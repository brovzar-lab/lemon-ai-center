# Diagnostic Toolkit

## Structural Diagnosis (Story Grid)

### Six Core Questions
1. **Genre?** External (Action, Horror, Crime, Thriller, Love, Comedy, Status) and Internal (Worldview, Morality, Status)
2. **Conventions and obligatory scenes present?** Check against genre list below
3. **POV consistent?** Right choice for this story?
4. **Objects of desire clear?** Conscious Want and Subconscious Need
5. **Controlling idea?** State in one sentence. If muddy, that's the problem.
6. **BH/MB/EP?** Beginning Hook (25%), Middle Build (50%), Ending Payoff (25%)

### Five Commandments (Every Story Unit)
1. **Inciting Incident** — Causal or coincidental event that upsets balance
2. **Progressive Complications** — Escalating difficulties (each WORSE than the last)
3. **Crisis** — Best Bad Choice or Irreconcilable Goods (must be a real dilemma)
4. **Climax** — The active choice
5. **Resolution** — New equilibrium

### Value Progressions
Every scene turns on a value spectrum: POSITIVE → CONTRARY → NEGATIVE → NEGATION OF NEGATION

- Action/Horror: Life → Unconsciousness → Death → Damnation
- Love: Love → Indifference → Hate → Self-Hatred
- Crime: Justice → Unfairness → Injustice → Tyranny
- Status: Success → Compromise → Failure → Selling Out

## Genre Obligations Quick Reference

### ACTION — Core Event: Hero at Mercy of Villain
- Hero who sacrifices for others
- Villain more powerful than hero
- Victims to be saved
- Hero's initial strategy fails
- All Is Lost moment
- Hero at Mercy of Villain scene
- Hero's sacrifice / final confrontation

### HORROR — Core Event: Victim at Mercy of Monster
- Monster that cannot be reasoned with
- Sin/transgression inviting the monster
- Confined/isolated setting
- Protagonist denies monster initially
- Escalating attacks revealing monster's nature
- Hero learns monster's rules/weakness
- Victim at Mercy of Monster scene
- False ending (monster returns)

### CRIME — Core Event: Exposure of Criminal (J'accuse)
- A crime committed or about to be
- Investigator (professional or amateur)
- Clues and red herrings
- "Making it personal" moment
- Ticking clock
- False accusation / wrong suspect
- J'accuse scene

### THRILLER — Core Event: Hero at Mercy of Villain (+ Damnation stakes)
- A MacGuffin
- Ticking clock
- Hero becomes the hunted
- Seemingly impossible odds
- Hero operates in villain's world

### LOVE — Core Event: Proof of Love
- Lovers meet
- External need forces togetherness
- Genuine obstacle to union
- First kiss/intimacy
- Breakup
- Proof of love scene (costs something)

### COMEDY — Core Event: Often Proof of Love
- Comedic premise with escalation
- Mistaken identity or miscommunication engine
- Commitment to a bad plan
- Public embarrassment
- Moment of truth — both funny and real

### STATUS/DRAMA — Core Event: Big Event granting or denying status
- Clear goal/ambition
- Mentor figure
- Rival
- Increasing cost of success
- Choice between ambition and integrity

## Character Diagnosis

### The Moral Component
1. **Moral Blind Spot** — False belief they cannot see, poisoning all choices
2. **Immoral Effect** — Blind spot in action, hurting others ON THE PAGE
3. **Dynamic Moral Tension** — Repeated moral choices through the middle forcing confrontation

### Active vs. Passive Test
- **PASSIVE:** Problem finds protagonist → reactive choice → reactive effect = SITUATION
- **ACTIVE:** Blind spot → immoral effect → problem → proactive choice → offer to change → refusal → loop = STORY

Flag passive protagonists aggressively. #1 development killer.

### Arc Elements (Weiland)
1. **The Lie** — Specific false belief driving behavior
2. **The Want** — External goal driven by the Lie (drives PLOT)
3. **The Need/Truth** — What they actually require (drives THEME)
4. **The Ghost/Wound** — Specific past event that created the Lie
5. **The Moral Blind Spot** — How the Lie manifests as harmful behavior

### Arc Types
- **Positive Change:** Lie → confrontation → Truth accepted
- **Flat:** Character holds Truth, changes the world
- **Negative:** Rejects Truth, embraces deeper Lie (tragedy)

### The Offer-Refusal Loop (Mushy Middle Fix)
Map 3-5 cycles through Act 2:
1. Immoral effect → consequence → proactive choice → offer to change → REFUSAL
2. Each refusal creates higher-stakes problem
3. Pattern of decline → Doom Moment (rock bottom)
4. Pattern of elevation: awakening → first step → reconnection → sees blind spot → chooses to heal

## Common Problems — Full Diagnosis Table

| Problem | Symptom | Root Cause | Fix |
|---|---|---|---|
| Sagging middle | Reader loses interest pp. 40-70, scenes feel episodic | Complications not progressive — different but not worse | List complications, rank severity, reorder ascending, cut any that don't top previous |
| Don't care about protagonist | No rooting interest, passive, inconsistent | No concrete Want OR protagonist doesn't make meaningful choices | Define specific Want with urgent stakes, ensure active pursuit with consequences |
| Ending doesn't work | Feels cheated, comes from nowhere, unearned | Climax doesn't deliver genre's core event OR resolution contradicts setup | Make climax the genre's obligatory core event, protagonist's choice creates outcome |
| Scenes feel pointless | Pretty but don't advance, dialogue clever but empty | Missing Five Commandments — no inciting incident, crisis, or value shift | Apply Five Commandments to every scene, cut scenes with no value shift |
| Feels generic | Predictable, "seen this before" | Conventions without innovation — delivering WHAT but not fresh HOW | Brainstorm 10 unexpected ways per convention, choose true to characters |
| No tension | Easy reading, forgettable, problems solved too easily | Crises aren't true dilemmas — one choice obviously better | Both options need real costs, audience should argue for either |
| Doesn't feel like genre | Marketed as thriller but not thrilling | Missing conventions or obligatory scenes | Checklist genre requirements, add missing, strengthen weak |
| Subplots disconnected | B-story feels like different movie | Not thematically linked to main plot | Every subplot explores same theme from different angle |
| Too long/short | 140+ or under 85 pages | Structural proportions off | 25/50/25 rule, cut non-escalating complications or add escalating ones |
| Contradictory notes | Feedback seems to contradict itself | Surface notes masking deeper structural problem | Don't address notes directly — find underlying issue via Six Core Questions |

## Revision Priority Order

When time is limited, fix in this order:
1. Global Genre (wrong = failure)
2. Obligatory Scenes (missing = genre failure)
3. Controlling Idea (unclear = meaningless)
4. Beginning Hook Inciting Incident (weak = lost audience)
5. Ending Payoff Climax (weak = unsatisfied audience)
6. All Is Lost Moment (missing = no catharsis)
7. Progressive Complications (flat = bored audience)
8. Individual Scenes (weak = forgettable)

Do not polish dialogue until structure is solid.

## Rewrite Pass Types

**Pass 1: Structure** — Every scene earns its place. Cut/merge scenes that don't turn.
**Pass 2: Character** — Arcs earned, protagonist drives action, antagonist has legitimate POV.
**Pass 3: Dialogue** — Distinct voices, subtext, no exposition dumps.
**Pass 4: Tension** — Stakes in every scene, urgency in every act.
**Pass 5: Theme** — Controlling idea through choices not speeches, subplots echo theme.
**Pass 6: Visual** — Show don't tell, cinematic action lines.
**Pass 7: Polish** — Typos, formatting, page count, presentation.
