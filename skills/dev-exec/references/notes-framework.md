# Notes Framework

## Notes Document Template

```
PROJECT:          [Title]
WRITER:           [Name]
DRAFT:            [Draft # / Date]
PASS TARGET:      [What this pass was supposed to accomplish]

EXECUTIVE SUMMARY:
[2-3 sentences. Did it hit pass goals? Trending direction?]
Pass Goals: Goal 1 [Met/Partial/Not Met] | Goal 2 [Met/Partial/Not Met]
Trajectory: [Improving / Flat / Declining]

WHAT'S WORKING (PROTECT):
• [Element — why it works. Tells writer what NOT to change.]

MUST FIX (This Pass — max 3):
Note 1: [Title]
  Problem: [Specific diagnosis + page refs]
  Root Cause: [Structural/character/genre reason]
  Impact: [What it does to reader experience]
  Prescription: Option A [solution] / Option B [solution] / Recommended: [which + why]

SHOULD FIX (This Pass If Possible — max 3):
Note 2: [Title — same format]

COULD FIX (Future Pass):
• [Brief note]

NEXT STEPS:
Next Pass Focus: [target] / Deliverable: [what] / Timeline: [when]
```

## Rules for Giving Notes

1. **Max 5 major notes per pass.** Fixing top 3 often resolves the next 5.
2. **Diagnose before prescribing.** "The villain isn't scary" is an observation. "There's no victim-at-mercy-of-monster scene" is a diagnosis.
3. **Give options, not orders.** "Option A or B — I lean toward A because..."
4. **Every note serves the North Star.** Check against the Project Identity Statement. If it contradicts, the note is wrong.
5. **Separate observation from solution.** Give both. Observations without solutions frustrate. Solutions without observations confuse.
6. **Don't rewrite in the notes.** Tell what to fix and suggest how. Don't write dialogue in notes documents.
7. **Acknowledge difficulty.** If a note requires structural overhaul, say so.

## Common Notes Mistakes

- **Contradictory:** "Make it faster" AND "add character development." Choose which matters this pass.
- **Vague:** "Something feels off in the middle." Diagnose before sending.
- **Taste-based:** "I don't like the sister." Not a note unless diagnosed (she's passive, no agenda, plot device).
- **Scope creep:** Pushing horror-comedy toward drama. Redefining the project, not developing it.
- **Cumulative:** Adding without removing. Every addition prompts: what do we cut?
- **Nostalgia:** "I liked Draft 2 better." Diagnose what was good and recapture it forward.

## Pass-Specific Notes Focus

**Structure:** Act breaks, Inciting Incident strength/timing, Midpoint shift, All Is Lost, Climax delivers genre core event, scene necessity, escalation.

**Character:** Active or passive? Blind spot visible? Arc milestones earned? Antagonist has legitimate POV? Supporting cast has own agendas?

**Genre:** Conventions present? Obligatory scenes delivered freshly? Core event is climax? Value progression tracked? Tone consistent?

**Dialogue:** Distinct voices (cover names test)? Subtext present? Exposition dramatized? Tactic changes within scenes?

**Polish:** Page count (110 +/- 5), action line economy, formatting, opening/closing images rhyme.
