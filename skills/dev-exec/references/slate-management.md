# Slate Management

## Slate Review Template

```
DATE:             [Date]
ACTIVE PROJECTS:  [Count]

PORTFOLIO HEALTH:
  Genre Mix:    Comedy [X] | Horror [X] | Drama [X] | Thriller [X] | Hybrid [X]
  Stage Mix:    Intake [X] | Pass 1 [X] | Pass 2 [X] | Pass 3+ [X] | Pre-Prod [X] | Incubation [X]
  Budget Mix:   Bajo [X] | Medio [X] | Alto [X]
  Assessment:   [Healthy / Imbalanced — specify]

PER-PROJECT STATUS:
  PROJECT: [Title]
  GENRE: [Genre]        TIER: [Budget]      WRITER: [Name]
  STAGE: [Intake / Pass 1 / Pass 2 / Pass 3 / Pre-Prod / Shelved]
  STATUS: [On Track / Needs Attention / At Risk / Stalled]
  SCORE: [/75]          TREND: [up/down/flat]
  LAST ACTION: [What happened]
  NEXT ACTION: [Specific step]
  DEADLINE: [Date]
  FLAG: [None / Writer Issue / Concept Issue / Budget Issue / Market Issue / Kill?]
```

## Status Definitions

**ON TRACK** — Progressing per Development Strategy. Pass goals being met. Score improving.

**NEEDS ATTENTION** — Hit a snag but recoverable. Writer missed deadline, pass partially met, new problem emerged. Requires strategy adjustment.

**AT RISK** — In danger of stalling or kill. Score declining, writer can't execute, concept problems emerging. Requires immediate intervention.

**STALLED** — Stopped making progress. Waiting on external factor, creative dead end, writer non-responsive. If stalled 30+ days, trigger kill criteria review.

## Monthly Decisions

For every project:
1. Still worth the investment? (Score trajectory vs. resource cost)
2. In the right stage? (Sometimes officially Pass 2 but functionally needs Pass 1 redo)
3. Right writer? (Failed to execute clear notes across two passes = consider change or kill)
4. Right timing? (Market shifts, similar releases)
5. Blocking other projects? (Stalled project consuming attention from healthier ones)

## Quarterly Decisions

- **Kill the bottom 20%.** Lowest scores, worst trajectories. If answer isn't clear "continue with plan," kill.
- **Rebalance genre portfolio.** Heavy on drama, light on comedy? Prioritize comedy intake.
- **Assess pipeline flow.** Healthy = projects at every stage. If everything's early development and nothing approaching pre-prod, pipeline is constrained.

## Portfolio Strategy

**Genre diversity is risk management.** Don't develop 10 of the same thing. Recommended 10-project mix: 3 comedies, 2 horror, 2 dramas, 2 hybrids, 1 wild card.

**Stagger development stages.** Avoid bottlenecks. Ideal monthly: 1-2 intake, 2-3 active development (different passes), 1 approaching handoff, 2-3 incubation.

**Allocate by potential, not history.** Sunk cost is not a reason to continue. Current potential is.

**Balance budget tiers.** All Alto = concentrated risk. All Bajo = limited upside. Mix tiers.

**Time-box everything.** No project in active development more than 12 months without reaching pre-prod or being killed.

## Health Indicators

**Healthy:** 60%+ On Track or Pre-Prod, scores trending up, genre mix matches target, pipeline has all stages, at least 1 kill per quarter.

**Unhealthy:** 30%+ Stalled or At Risk, scores flat/declining, genre lopsided, pipeline clogged, no kills in 6+ months.

**Crisis:** 50%+ Stalled, nothing approaching Pre-Prod, multiple projects past 12 months, writers burned out. Response: Stop all development, run kill criteria on everything, kill below 35, rebuild from intake.
